## Authentication

<Card horizontal title="Authentication Guide" icon="key" href="/authentication">
  Learn more about securing your API requests with authentication.
</Card>

All API requests require an API key in the `x-api-key` header:

<CodeGroup>

```typescript TypeScript
const headers = {
  'Content-Type': 'application/json',
  'x-api-key': process.env.API_KEY,
}

// Test authentication
const response = await fetch('https://api.orders.sdigital2.com/v1/users', {
  headers,
})

if (!response.ok) {
  throw new Error('Authentication failed')
}

const { data: user } = await response.json()
console.log('Authenticated as:', user.email)
```

```python Python
import requests
import os

headers = {
    'Content-Type': 'application/json',
    'x-api-key': os.getenv('API_KEY')
}

# Test authentication
response = requests.get(
    'https://api.orders.sdigital2.com/v1/users',
    headers=headers
)

response.raise_for_status()
user = response.json()['data']
print(f"Authenticated as: {user['email']}")
```

</CodeGroup>

## Creating Recipients

<Card
  horizontal
  title="Recipients Guide"
  icon="user-group"
  href="/guides/recipients"
>
  Learn more about creating and managing recipients.
</Card>

Before creating orders, set up a recipient:

<CodeGroup>

```typescript TypeScript
// Create a crypto recipient
const cryptoRecipient = {
  type: 'crypto',
  token: 'USDC_ETH',
  address: '0x742d35Cc6634C0532925a3b844Bc454e4438f44e',
  business_details: {
    name: 'Acme Corporation',
  },
}

const response = await fetch('https://api.orders.sdigital2.com/v1/recipients', {
  method: 'POST',
  headers,
  body: JSON.stringify(cryptoRecipient),
})

const { data: recipient } = await response.json()
console.log('Created recipient:', recipient.id)
```

```python Python
# Create a crypto recipient
crypto_recipient = {
    'type': 'crypto',
    'token': 'USDC_ETH',
    'address': '0x742d35Cc6634C0532925a3b844Bc454e4438f44e',
    'business_details': {
        'name': 'Acme Corporation'
    }
}

response = requests.post(
    'https://api.orders.sdigital2.com/v1/recipients',
    headers=headers,
    json=crypto_recipient
)

recipient = response.json()['data']
print(f"Created recipient: {recipient['id']}")
```

</CodeGroup>

## Creating Orders

<Card horizontal title="Orders Guide" icon="file-invoice" href="/guides/orders">
  Learn more about creating and managing orders.
</Card>

Create an order using your recipient:

<CodeGroup>

```typescript TypeScript
const order = {
  from: {
    currency: 'USD',
    amount: 1000,
  },
  to: {
    currency: 'USDC_ETH',
    recipient_id: recipient.id,
  },
  is_third_party: false,
}

const response = await fetch('https://api.orders.sdigital2.com/v1/orders', {
  method: 'POST',
  headers,
  body: JSON.stringify(order),
})

const { data: createdOrder } = await response.json()
console.log('Created order:', createdOrder.id)
```

```python Python
order = {
    'from': {
        'currency': 'USD',
        'amount': 1000
    },
    'to': {
        'currency': 'USDC_ETH',
        'recipient_id': recipient['id']
    },
    'is_third_party': False
}

response = requests.post(
    'https://api.orders.sdigital2.com/v1/orders',
    headers=headers,
    json=order
)

created_order = response.json()['data']
print(f"Created order: {created_order['id']}")
```

</CodeGroup>

## Confirming Orders

<Card
  horizontal
  title="Order Confirmation Guide"
  icon="check-circle"
  href="/guides/orders/confirmation"
>
  Learn more about the order confirmation process.
</Card>

For crypto orders, confirm with the transaction hash:

<CodeGroup>

```typescript TypeScript
const orderId = createdOrder.id
const confirmation = {
  transaction_hash:
    '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef',
}

const response = await fetch(
  `https://api.orders.sdigital2.com/v1/orders/${orderId}/confirm`,
  {
    method: 'PUT',
    headers,
    body: JSON.stringify(confirmation),
  }
)

const { data: confirmedOrder } = await response.json()
console.log('Order confirmed:', confirmedOrder.status)
```

```python Python
order_id = created_order['id']
confirmation = {
    'transaction_hash': '0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef'
}

response = requests.put(
    f'https://api.orders.sdigital2.com/v1/orders/{order_id}/confirm',
    headers=headers,
    json=confirmation

print(f"Order confirmed: {confirmed_order['status']}")
```

## Error Handling

Implement robust error handling:

<CodeGroup>

```typescript TypeScript
try {
  const response = await fetch('https://api.orders.sdigital2.com/v1/orders', {
    method: 'POST',
    headers,
    body: JSON.stringify(order),
  })

  if (!response.ok) {
    const error = await response.json()
    throw new Error(error.error.message)
  }

  const { data } = await response.json()
  console.log('Success:', data)
} catch (error) {
  console.error('Error:', error.message)
}
```

```python Python
try:
    response = requests.post(
        'https://api.orders.sdigital2.com/v1/orders',
        headers=headers,
        json=order
    )
    response.raise_for_status()

    data = response.json()['data']
    print('Success:', data)
except requests.exceptions.RequestException as error:
    print('Error:', str(error))
```

</CodeGroup>
